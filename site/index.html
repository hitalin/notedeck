ダウンロード GitHub

機能

全鯖を一画面で

複数サーバーに同時ログイン。HTL・通知・リスト・チャンネルをカラムに並べて、タブ切り替えなしで全体を把握。

フォークもそのまま使える

Misskey 本家に加え、Firefish・CherryPick など主要フォークに対応。サーバー広告もちゃんと表示します。

ブラウザから解放される

システムトレイ常駐、グローバルホットキー、マルチウィンドウ。タブに埋もれない、ネイティブアプリならではの快適さ。

ノートを手元に蓄積

流れたノートもローカル DB に残る。全文検索で過去の投稿をすぐ見つけられる。設定はテキストファイルだから丸ごとバックアップも簡単。

軽い。速い。

Tauri v2 + Rust バックエンドで省メモリ・高速起動。重いクライアントに慣れた人ほど違いを感じるはず。

オープンソース

AGPL-3.0 で全コード公開。欲しい機能があれば PR も歓迎。Sponsors で開発を支援できます。
