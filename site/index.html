マルチカラムデッキ

タイムライン・通知・リストなどを自由に並べて、一画面で把握。カラム幅はドラッグで調整可能。

🌐

マルチサーバー

複数の Misskey サーバーに同時ログイン。ひとつのデッキですべてのアカウントを管理。

🔀

フォーク対応

Misskey 本家に加え、主要フォーク（Firefish, CherryPick など）にも対応。

📱

クロスプラットフォーム

Windows, macOS, Linux, Android に対応。Tauri v2 による軽量ネイティブアプリ。

🔓

オープンソース

AGPL-3.0 ライセンスで公開。コードは GitHub で誰でも閲覧・貢献可能。

⚡

軽量・高速

Rust バックエンドで省メモリ・高速起動。Electron 系クライアントとは一線を画すパフォーマンス。
