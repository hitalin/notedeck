NoteDeck

複数の Misskey サーバーをひとつのデッキで。

ダウンロード GitHub

NoteDeck とは

NoteDeck は、Misskey 系サーバー向けのマルチカラム・デッキクライアントです。 複数のサーバーに同時ログインして、タイムライン・通知・リストなどを一画面に並べて使えます。 Tauri v2 + Rust で構築されており、Electron 系クライアントより軽量に動作します。

機能

  • マルチカラム タイムライン・通知・リスト・チャンネルなどを自由に並べて表示。幅はドラッグで調整可能
  • マルチサーバー 複数の Misskey サーバーに同時ログイン。ひとつのデッキですべてのアカウントを管理
  • フォーク対応 Misskey 本家に加え、Firefish・CherryPick などの主要フォークに対応
  • クロスプラットフォーム Windows / macOS / Linux / Android で動作
  • 軽量・高速 Rust バックエンド（Tauri v2）で省メモリ・高速起動
  • プロファイル カラム構成をプロファイルとして保存・切り替え
  • カスタマイズ可能なナビバー よく使うカラムをナビバーに配置してワンタップで切り替え
  • オープンソース AGPL-3.0 ライセンスで公開。誰でもコードを閲覧・貢献可能

パッケージマネージャー

winget
winget install Hitalin.NoteDeck
AUR
yay -S misskey-notedeck-bin
Nix
nix run github:hitalin/notedeck
Windows / Android ではインストール時にセキュリティ警告が表示されることがあります。 これはアプリがコード署名されていないためで、マルウェアではありません。 ソースコードは公開されており、ビルドは GitHub Actions で自動化されています。
